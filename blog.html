1. Blog Questions Discuss the scope of var, let, and const

Var: var is a functional scope variable. if we use var without function than we can access var from any were like global or block scope

Let: let is a block scope variable. if we use let on globally than we can access from any were like function or block scope. and if we use let in function or block scope we cant access the let out side of function or block scope.

Const: const is a block scop like let.
