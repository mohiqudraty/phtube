1. Discuss the scope of var, let, and const

Var: var is a functional scope. if we use var without function than we can access var from any where.

Let: let is a block scope variable. if we use let on globally than we can access from any where . and if we use let in function or block scope we cant access the let out side of function or block scope.

Const: const is a block scop like let. but if we use const than we can not reassign the value of const .




2. Tell us the use cases of null and undefined

Null: when we want indicate a variable they have no meaningfully value or property. and when we want to remove reference to an object or others element.
